Mobile Optimization Considerations
* Headlines: Keep to 8-12 words maximum; use short, punchy subheadlines
* CTAs: Buttons must be at least 44px tall, easy to tap with thumb; place CTA above the fold on mobile and at least 3 more times throughout
* Testimonials: Use vertical scrollable carousel or single quote with name; avoid side-by-side on small screens
* Bullet Lists: Convert to stacked (one bullet per line) for readability
* Spacing: Generous padding between sections; avoid cluttered layouts
* Form Fields: Minimal required fields (name, email, phone, location preference, service interest); use dropdowns for location and service
* Images: Compress for fast load; use lazy loading; ensure before/after images are clear on small screens
* Load Speed: Optimize for <3 seconds; suggest using CDN for images; minimize JavaScript

Technical Implementation Guide
Tracking Pixel Placement
* Facebook Pixel: Place header code on the landing page; set up standard events (Lead, ViewContent, CompleteRegistration).
* Google Tag: Install via Google Tag Manager; trigger conversion events on "Thank You" page (consultation booked).
* Google Ads: Use remarketing lists based on page visitors (e.g., "Viewers of Hero section" for awareness retargeting).
* UTM Parameters: Track all ad clicks with source, medium, campaign, content, and term.


CRM Integration Points
* Lead Capture Form: Name, email, phone, location preference, primary concern (dropdown), how they heard about us.
* Zapier/Integromat: Connect form submission to Zenoti for automated lead creation.
* Segment Tags: Tag leads by treatment interest (Laser, Injectables, Body, Weight Loss) for personalized follow-up.
* SMS Automation: Use Twilio or similar for appointment reminders (48h and 24h before).


Page Speed Optimization Checklist
* Compress all images to <200KB (use WebP format).
* Implement lazy loading for below-fold images.
* Minify CSS, JavaScript, HTML.
* Use a CDN for static assets.
* Reduce number of HTTP requests (combine files).
* Enable browser caching.
* Remove any unused plugins or scripts.


Analytics Setup
* Google Analytics 4: Enable enhanced measurement for scroll depth, outbound clicks, site search, video engagement.
* Conversion Tracking: Define "Consultation Booked" as a key event; set up goals for "Form Submitted" and "Thank You Page Viewed."
* Heatmaps: Use Hotjar or CrazyEgg to track mouse movement, scroll maps, and click zones for UX optimization.
